## 信令

### 加入房间
1. 请求
```
{
    "cmd":"join",
    "roomId":"45345",
    "uid":"32432r"
}
```

2. 返回
```
{
    "cmd": "resp-join",
    "roomId":"45345",
    "uid":"32432r",
    "members":[{"uid":"33233d"}]
}
```

### 发布流
1. 请求
```
{
    "cmd":"publish",
    "roomId":"45345",
    "uid":"32432r",
    "sdp":""
}
```
2. 返回
```
{
    "cmd":"resp-publish",
    "roomId":"45345",
    "uid":"32432r",
    "ret_code": 0,
    "sdp":""
}
```

### 拉流
1. 请求
```
{
    "cmd": "pullstream",
    "roomId": "45345",
    "uid": "32432r",
    "sdp":""
}
```

2. 返回
```
{
    "cmd":"resp-pullstream",
    "roomId":"45345",
    "uid":"32432r",
    "ret_code": 0,
    "sdp":""
}
```
